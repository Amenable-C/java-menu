## 🚩기능 목록
- [ ] 코치들의 이름을 입력 받는다.
    - [ ] 코치의 이름은 최소 2글자, 최대 4글자이다.
    - [ ] 코치는 최소 2명, 최대 5명이다.
- [ ] 각 코치들의 먹지 못하는 메뉴를 입력받는다.
    - [ ] 먹지 못하는 메뉴가 없으면 빈 칸을 입력받는다.
- [ ] 한 주(월요일~화요일)의 메뉴 카테고리를 만든다.
    - [ ] 한 주에 같은 카테고리는 최대 2회까지만 고를 수 있다.
- [ ] 각 코치들에게 카테고리에 맞는 메뉴를 추천해준다.
    - [ ] 한 주에 중복되지 않는 메뉴를 추천해야 한다.
